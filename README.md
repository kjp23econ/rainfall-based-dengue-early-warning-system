Full reproducible R scripts for a Dengue Early Warning System in the Philippines, including weekly rainfall, dengue cases, pediatric admissions, and case fatality rate analysis. Figures 1–10 included with rolling thresholds and pooled Youden analysis. Data cannot be shared due to source restrictions.

## Overview

This repository contains the full reproducible R scripts for a Dengue Early Warning System in the Philippines. It includes:

- Weekly dengue cases and rainfall analysis
- Pediatric admissions and case fatality rate (CFR) tracking
- Rolling case thresholds and pooled Youden index computation
- Figures 1–10 for visualizing early warning signals and model outputs

> **Note:** The raw datasets cannot be shared publicly due to the terms and conditions of the data sources. Users should replace the Excel sheets with their own local copies to replicate analyses.
